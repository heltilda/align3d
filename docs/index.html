align3d

A program for reconstructing chromosome conformation from discrete loci





align3d is a reconstruction tool for recovering DNA conformational information from a) a set of labeled loci, and b) a microscope image of those labels.

The program is written on C++, so should compile on any machine with a C++ compiler.

Reference: Ross and Wiggins, PRE 2012
